# Workflows

STMNA Desk includes internal automation workflows for vault operations
and knowledge management. These workflows are part of the private STMNA OS
operational layer and are not included in this repository.

For importable workflow examples, see:
- [STMNA Signal workflows](https://github.com/stmna-io/stmna-signal/tree/main/workflows) — content ingestion and processing pipelines
- [STMNA Voice](https://github.com/stmna-io/stmna-voice) — speech-to-text pipeline
